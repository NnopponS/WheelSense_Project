%% Abstract - Thai (ภาษาวิชาการ)
%%================================================

งานวิจัยครั้งนี้มีวัตถุประสงค์เพื่อพัฒนาระบบต้นแบบสภาพแวดล้อมอัจฉริยะสำหรับผู้ใช้งานเก้าอี้รถเข็นในพื้นที่ภายในอาคาร
โดยมุ่งเสริมความปลอดภัย ความสะดวก และความเป็นอิสระในการดำรงชีวิตประจำวัน
แกนกลางของการออกแบบคือการประกอบ \textbf{สายงานข้อมูลแบบ end-to-end} ที่ผู้ดูแลสามารถนำไปใช้ติดตามสถานะและประกอบการตัดสินใจได้อย่างใกล้เคียงเวลาจริง (near real-time)

ส่วนประกอบของระบบประกอบด้วยอุปกรณ์บนตัวเก้าอี้รถเข็น (M5StickC Plus2) สำหรับรวบรวมข้อมูลการเคลื่อนไหวจาก IMU สถานะแบตเตอรี่ และผลการสแกน BLE/RSSI เพื่อใช้ในงานระบุตำแหน่งภายในอาคาร
การระบุตำแหน่งดำเนินการด้วยแนวทาง RSSI fingerprinting ร่วมกับ BLE beacon โดยประมวลผลบนเซิร์ฟเวอร์ด้วยอัลกอริทึม KNN
กำหนดโปรโตคอลการเก็บ fingerprint และชุดพารามิเตอร์อ้างอิงเพื่อให้การทำซ้ำและการถ่ายทอดผลเป็นไปได้อย่างมีระเบียบ

เพื่อเสริมการรับรู้บริบทสำหรับผู้ดูแล งานวิจัยนี้ผสานโหนดกล้อง \texttt{Node\_Tsimcam} (ESP32-S3) สำหรับส่ง \textbf{ภาพนิ่ง (photo)} ผ่าน MQTT แบบแบ่งชิ้นส่วน (chunked payload) พร้อมแนวทางสำรองด้วย HTTP เมื่อสภาวะเครือข่ายไม่เอื้ออำนวย
โดยไม่ได้นำสตรีมภาพมาเป็นฟีเจอร์หลักของโมเดลการระบุตำแหน่ง

ด้านสรีรศาสตร์ ระบบบูรณาการข้อมูลจากอุปกรณ์ Polar Verity Sense เพื่อการติดตามสัญญาณชีพ (เช่น HR/PPG) เข้าสู่สายงาน telemetry เดียวกัน
และมีตรรกะการแจ้งเตือนเบื้องต้นแบบอิงกฎ (rule-based) จากสัญญาณ accelerometer

ชั้นแอปพลิเคชันประกอบด้วยแดชบอร์ดเว็บสำหรับผู้ดูแล และแอปพลิเคชันบนอุปกรณ์เคลื่อนที่แบบข้ามแพลตฟอร์ม (React Native + Expo) รองรับบริบทการใช้งานแบบพกพาและโหมด \textit{walking}
พร้อมการเชื่อมต่อกับ Home Assistant เพื่อการทำงานอัตโนมัติที่สอดคล้องกับบริบท (context-aware automation)

นอกจากนี้ งานวิจัยประยุกต์ใช้ local LLM (Gemma 4B) ผ่าน MCP (Model Context Protocol) ภายใต้กรอบการทำงานแบบเสนอ--ยืนยัน--ดำเนินการ (propose--confirm--execute)
เพื่อช่วยสรุปและตีความข้อมูลภายใต้การจำกัดขอบเขตเครื่องมือที่ระบบอนุญาตให้เรียกใช้ได้

ผลการพัฒนาสะท้อนความเป็นไปได้ของระบบต้นแบบตั้งแต่การรับสัญญาณ การจัดเก็บ การคำนวณผล ไปจนถึงการแสดงผลแบบบูรณาการ
ข้อจำกัดที่ยังต้องพัฒนาต่อไป ได้แก่ ความสอดคล้องของเวลา (การจัดเรียง timestamp) ระหว่างองค์ประกอบย่อยของระบบ และความซับซ้อนของการติดตั้งภาคสนาม
ซึ่งเป็นแนวทางสำคัญสำหรับการพัฒนาต่อยอดเพื่อการนำไปใช้ในที่พักอาศัย โรงพยาบาล และสถานประกอบการดูแล
