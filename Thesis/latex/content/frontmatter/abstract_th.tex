โครงงานนี้พัฒนาระบบต้นแบบสภาพแวดล้อมอัจฉริยะสำหรับผู้ใช้เก้าอี้รถเข็น เพื่อเพิ่มความปลอดภัย ความสะดวก และความเป็นอิสระในการใช้ชีวิตภายในอาคาร โดยมุ่งสร้าง \textbf{data pipeline แบบ end-to-end} ที่ผู้ดูแลสามารถใช้ติดตามและตัดสินใจได้ในเวลาใกล้เคียงจริง (near real-time)
ระบบประกอบด้วยอุปกรณ์บนเก้าอี้รถเข็น (M5StickC Plus2) สำหรับรวบรวมข้อมูลการเคลื่อนที่จาก IMU และข้อมูลแบตเตอรี่ รวมถึงการสแกน BLE/RSSI เพื่อใช้ในงานระบุตำแหน่งภายในอาคาร
การระบุตำแหน่งใช้แนวทาง RSSI fingerprinting ร่วมกับ BLE beacon และประมวลผลบนเซิร์ฟเวอร์ด้วยอัลกอริทึม KNN โดยกำหนดโปรโตคอลการเก็บ fingerprint และพารามิเตอร์อ้างอิงเพื่อให้ทำซ้ำได้
เพื่อสนับสนุนการรับรู้บริบทของผู้ดูแล ระบบมีโหนดกล้อง \texttt{Node\_Tsimcam} (ESP32-S3) ส่ง \textbf{ภาพนิ่ง (photo)} ผ่าน MQTT แบบ chunked payload และมีแนวทางสำรองด้วย HTTP เมื่อจำเป็น โดยไม่ใช้ภาพเป็นฟีเจอร์หลักของโมเดล localization
ด้านสรีรศาสตร์ ระบบผสานข้อมูลจาก Polar Verity Sense สำหรับการติดตามสัญญาณชีพ (เช่น HR/PPG) เข้าสู่เส้นทาง telemetry เดียวกัน และมีตรรกะแจ้งเตือนเบื้องต้นแบบ rule-based จากสัญญาณ accelerometer
ชั้นแอปพลิเคชันประกอบด้วย web dashboard สำหรับผู้ดูแลและแอปมือถือแบบข้ามแพลตฟอร์ม (React Native + Expo) สำหรับบริบทการใช้งานพกพาและโหมด \textit{walking} พร้อมการบูรณาการกับ Home Assistant เพื่อ context-aware automation
นอกจากนี้ระบบประยุกต์ใช้ local LLM (Gemma 4B) ผ่าน MCP (Model Context Protocol) ภายใต้กรอบการยืนยันก่อนกระทำ (propose--confirm--execute) เพื่อช่วยสรุปและตีความข้อมูลโดยจำกัดขอบเขตเครื่องมือที่เรียกได้
ผลการพัฒนายืนยันความเป็นไปได้ของระบบต้นแบบตั้งแต่การรับสัญญาณ การจัดเก็บ การคำนวณผล และการแสดงผลแบบบูรณาการ โดยข้อจำกัดที่ยังต้องปรับปรุง ได้แก่ ความสอดคล้องของเวลา (timestamp alignment) ระหว่างองค์ประกอบและความซับซ้อนของการติดตั้งภาคสนาม ซึ่งเป็นแนวทางพัฒนาต่อเพื่อการนำไปใช้ในบ้าน โรงพยาบาล และสถานดูแล
