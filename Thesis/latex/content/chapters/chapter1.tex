\section{ที่มาและความสำคัญ}
\label{sec:ch1_significance}

จากสถานการณ์คนพิการในประเทศไทย ปี พ.ศ. 2567 \cite{MSDHS_Disability_2024} พบว่ามีประชากรผู้พิการกว่า 2.2 ล้านคน โดยเฉพาะผู้พิการทางการเคลื่อนไหว ภาครัฐจึงได้มีนโยบายส่งเสริมคุณภาพชีวิตหลายด้าน ทั้งการสร้างอาชีพผ่านรายงานการจ้างงานคนพิการ \cite{DEP_Employment_2024} การสนับสนุนสิทธิและการใช้ชีวิตอิสระตามอนุสัญญาว่าด้วยสิทธิคนพิการ (CRPD) \cite{UN_CRPD_2008} รวมถึงการประยุกต์ใช้เทคโนโลยีเพื่อการดำรงชีวิตอิสระสำหรับผู้สูงอายุและผู้พิการ (AAL) \cite{Rashidi_AAL_2023} เพื่อลดอุปสรรคในการดำเนินชีวิต นอกจากนี้ยังมีการส่งเสริมการมีส่วนร่วมทางสังคมตามแผนพัฒนาการกีฬาแห่งชาติ ฉบับที่ 6 \cite{MOTS_SportsPlan_2022} และการสร้างสภาพแวดล้อมที่ปราศจากอุปสรรคตามแผนพัฒนาคุณภาพชีวิตคนพิการแห่งชาติ ฉบับที่ 5 \cite{DEP_DisabilityPlan_2023}

ผู้ใช้เก้าอี้รถเข็นและผู้สูงอายุในสถานดูแล เช่น บ้านพักคนชราวาสนะเวศม์ ซึ่งเป็นบริบทศึกษาของโครงงานนี้ ต้องพึ่งพาผู้ดูแลและเผชิญข้อจำกัดด้านการเคลื่อนที่และความปลอดภัย

จากการลงพื้นที่สำรวจอาคารและพื้นที่บริการหลัก พบว่ามีผู้พักอาศัยกระจายอยู่ในหลายอาคาร ทำให้ผู้ดูแลไม่สามารถทราบตำแหน่งและสภาพของผู้ใช้เก้าอี้รถเข็นทุกคนได้ทันเวลา ซึ่งอาจนำไปสู่อุบัติเหตุ เช่น การหกล้ม \cite{Sterling2016FallsInCareHomes}

    \label{fig:ch1-fig02-facility-context}

นอกจากนี้ การทำงานของผู้ดูแลยังมีข้อจำกัดจากการบันทึกข้อมูลแบบแอนะล็อก เช่น กระดานหรือกระดาษ ซึ่งเสี่ยงต่อการสูญหายและไม่สามารถนำมาวิเคราะห์แนวโน้มสุขภาพได้ ปริมาณงานที่มากยังนำไปสู่ความล้าจากการแจ้งเตือน (alert fatigue) \cite{Gates2018AlertFatigue}

แม้ปัจจุบันจะมีระบบอัตโนมัติในบ้านและระบบระบุตำแหน่งภายในอาคาร \cite{Home_Automation_Integration_2023, Indoor_GPS_Limitations, UWB_Accuracy_Spec} แต่มักไม่สอดคล้องกับพฤติกรรมของผู้ใช้เก้าอี้รถเข็น มีความซับซ้อน หรือมีต้นทุนสูง โครงงานนี้จึงเสนอการพัฒนาแพลตฟอร์ม WheelSense ซึ่งเป็นระบบสภาพแวดล้อมอัจฉริยะแบบบูรณาการ ที่เชื่อมโยงอุปกรณ์บนเก้าอี้รถเข็น อุปกรณ์สวมใส่ โหนดภายในอาคาร และเซิร์ฟเวอร์ขอบ เพื่อสนับสนุนการทำงานของผู้ดูแลและเพิ่มความปลอดภัยให้แก่ผู้ใช้งานอย่างมีประสิทธิภาพ

\section{วัตถุประสงค์}
\label{sec:ch1_objectives}

\begin{enumerate}
    \item เพื่อพัฒนาระบบต้นแบบสภาพแวดล้อมอัจฉริยะสำหรับผู้ใช้เก้าอี้รถเข็นที่สามารถรับข้อมูลการเคลื่อนไหว
    สัญญาณชีพ และบริบทของพื้นที่ภายในอาคารได้อย่างต่อเนื่อง
    \item เพื่อพัฒนาระบบระบุตำแหน่งภายในอาคารระดับโซนหรือห้อง โดยอาศัยสัญญาณ BLE และการประมวลผล
    บนเซิร์ฟเวอร์
    \item เพื่อพัฒนาแพลตฟอร์มซอฟต์แวร์สำหรับจัดเก็บ ประมวลผล และแสดงผลข้อมูลแก่ผู้ใช้งานตามบทบาท
    ผ่านเว็บ Dashboard และแอปพลิเคชันบนอุปกรณ์เคลื่อนที่
    \item เพื่อบูรณาการผู้ช่วยปัญญาประดิษฐ์ภายใต้กรอบเสนอ--ยืนยัน--ดำเนินการ เพื่อช่วยสรุปข้อมูลและ
    สนับสนุนการตัดสินใจของผู้ดูแลอย่างปลอดภัย
    \item เพื่อประเมินระบบต้นแบบในด้านความถูกต้องของการระบุตำแหน่ง ความเสถียรของการไหลของข้อมูล
    ประสิทธิภาพของส่วนบริการ และประสบการณ์ผู้ใช้ของผู้ดูแล
\end{enumerate}

\section{ขอบเขตงานวิจัย}
\label{sec:ch1_scope}

\begin{enumerate}
    \item พัฒนาชุดอุปกรณ์ภาคสนามสำหรับติดตั้งบนเก้าอี้รถเข็นและอุปกรณ์สวมใส่ที่ใช้เก็บข้อมูลการเคลื่อนไหว
    และสัญญาณชีพพื้นฐาน
    \item พัฒนาระบบระบุตำแหน่งภายในอาคารระดับโซนหรือห้อง โดยใช้ข้อมูล BLE RSSI และข้อมูลอ้างอิงจาก
    พื้นที่ที่กำหนด
    \item พัฒนาเว็บ Dashboard และแอปพลิเคชันบนอุปกรณ์เคลื่อนที่สำหรับการใช้งานตามบทบาทของผู้ดูแล
    ผู้ปฏิบัติงาน และผู้เกี่ยวข้อง
    \item บูรณาการผู้ช่วยปัญญาประดิษฐ์แบบภายในระบบ โดยจำกัดการกระทำที่มีผลเปลี่ยนสถานะให้อยู่ภายใต้
    ขั้นตอนยืนยันก่อนดำเนินการ
    \item ประเมินระบบต้นแบบภายใต้สภาพแวดล้อมทดลองและการใช้งานสาธิตในบริบทสถานดูแล โดยไม่มุ่งหมาย
    เป็นการทดลองทางคลินิกหรือการรับรองเป็นเครื่องมือแพทย์
\end{enumerate}

\section{ประโยชน์ที่คาดว่าจะได้รับ}
\label{sec:ch1_benefits}

\textbf{ประโยชน์ต่อสังคม}
\begin{enumerate}
    \item ช่วยยกระดับคุณภาพการดูแลผู้สูงอายุและผู้ใช้เก้าอี้รถเข็นในสถานดูแล โดยเพิ่มความสามารถในการเฝ้าระวัง
    ความปลอดภัย การติดตามตำแหน่ง และการรับรู้เหตุผิดปกติได้รวดเร็วยิ่งขึ้น
    \item ช่วยลดภาระงานซ้ำซ้อนของผู้ดูแลและสนับสนุนการประสานงานข้ามเวรด้วยข้อมูลที่ค้นย้อนกลับได้
    ซึ่งเอื้อต่อการจัดการทรัพยากรบุคคลอย่างมีประสิทธิภาพมากขึ้น
\end{enumerate}

\textbf{องค์ความรู้ที่ได้รับ}
\begin{enumerate}
    \item ได้องค์ความรู้ด้านการออกแบบระบบสภาพแวดล้อมอัจฉริยะที่บูรณาการอุปกรณ์ภาคสนาม บริการส่วนกลาง
    และส่วนติดต่อผู้ใช้เข้าด้วยกันในบริบทการดูแล
    \item ได้องค์ความรู้ด้านการประยุกต์ใช้ผู้ช่วยปัญญาประดิษฐ์อย่างปลอดภัยในงานเชิงปฏิบัติการที่เกี่ยวข้องกับ
    ข้อมูลผู้พักอาศัยและการตัดสินใจของผู้ดูแล
\end{enumerate}
