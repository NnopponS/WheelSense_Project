# WheelSense Agent Guide

## Project Skills

This project provides skills in `.agent/skills/` (also linked at `.cursor/skills/` for Cursor). Use them before starting tasks:

- **nextjs-frontend** – Next.js 16, TypeScript, Tailwind v4, Zustand, React Query
- **fastapi-backend** – FastAPI, aiosqlite, MQTT, Home Assistant
- **docker-deployment** – Docker Compose stack
- **mqtt-protocol** – MQTT topic structure, M5StickCPlus2 format
- **systematic-debugging** – Root cause analysis, no band-aid fixes
- **error-handling** – Resilience patterns
- **testing** – Vitest (frontend), pytest (backend)

Read `SKILL.md` from the relevant skill directory before implementing.
