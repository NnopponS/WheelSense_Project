# Supplementary material (outcome)

This folder contains train/test data, outcome tables, model descriptions, and a Streamlit viewer for the three BLE RSSI evaluation cases. To run the viewer from the project root: `pip install -r outcome/requirements.txt` then `streamlit run outcome/app.py`. If you only have this folder (e.g. after unzipping), run from inside it: `pip install -r requirements.txt` then `streamlit run app.py`. See [Info.md](Info.md) for experiment details and [ForPaper.md](ForPaper.md) for reproducibility notes.
